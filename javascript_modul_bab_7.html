DOM Manipulation Lanjutan

1. Menambahkan Elemen Baru

let list = document.getElementById('daftar');
let itemBaru = document.createElement('li');
itemBaru.innerText = 'Item Baru';
list.appendChild(itemBaru);

2. Menghapus Elemen

let item = document.getElementById('hapus');
item.remove();

3. Traversing DOM

  • parentElement - Mengakses elemen induk
  • children - Mengakses semua anak elemen
  • nextElementSibling - Mengakses elemen setelahnya
  • previousElementSibling - Mengakses elemen sebelumnya
let item = document.getElementById('item1');
console.log(item.parentElement);
console.log(item.nextElementSibling);

4. Manipulasi Class

let kotak = document.getElementById('kotak');
kotak.classList.add('merah'); // Menambah class
kotak.classList.remove('biru'); // Menghapus class
kotak.classList.toggle('hijau'); // Menambah/menghapus class

Latihan:

  • Buat tombol "Tambah Item" yang akan menambahkan elemen <li> baru ke dalam daftar.
  • Buat tombol "Hapus Item" yang akan menghapus item terakhir dari daftar.
  • Buat tombol "Toggle Warna" yang akan menambahkan atau menghapus class warna pada sebuah div.
