Variabel, Tipe Data, dan Operator

1. Variabel dalam JavaScript

Variabel digunakan untuk menyimpan data. Ada tiga cara mendeklarasikan variabel:

  • var: Variabel lama, jarang digunakan pada modern JavaScript.
  • let: Variabel yang dapat diubah.
  • const: Variabel yang nilainya tetap (konstan).
let nama = 'Fikri';
const umur = 25;
var alamat = 'Bandung';

2. Tipe Data

  • String: "Teks"
  • Number: 1, 2.5, 100
  • Boolean: true, false
  • Null: Kosong
  • Undefined: Belum diberi nilai
let nama = "Fikri"; // String
let umur = 25;      // Number
let aktif = true;   // Boolean
let kosong = null;  // Null
let belum;          // Undefined

3. Operator

  • Aritmatika: +, -, *, /, %
  • Perbandingan: ==, ===, !=, <, >, <=, >=
  • Logika: &&, ||, !
let a = 10;
let b = 5;
console.log(a + b); // Penjumlahan
console.log(a > b); // Perbandingan
console.log(a > 0 && b > 0); // Logika

Latihan:

  • Buat variabel untuk menyimpan nama, umur, dan status pekerjaan kamu.
  • Buat operasi matematika sederhana: penjumlahan, pengurangan, perkalian, dan pembagian.
  • Gunakan operator perbandingan untuk membandingkan dua angka.
