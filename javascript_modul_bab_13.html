Local Storage dan Session Storage

1. Pengertian Local Storage dan Session Storage

  • Local Storage: Menyimpan data di browser tanpa batas waktu.
  • Session Storage: Menyimpan data selama sesi browser aktif.

2. Local Storage

// Menyimpan data
localStorage.setItem('nama', 'Fikri');

// Mengambil data
let nama = localStorage.getItem('nama');
console.log(nama);

// Menghapus data
localStorage.removeItem('nama');

// Menghapus semua data
localStorage.clear();

3. Session Storage

// Menyimpan data
sessionStorage.setItem('status', 'Login');

// Mengambil data
let status = sessionStorage.getItem('status');
console.log(status);

// Menghapus data
sessionStorage.removeItem('status');

// Menghapus semua data
sessionStorage.clear();

4. Menyimpan dan Mengambil Data Objek

let user = { nama: 'Fikri', umur: 25 };

// Menyimpan object dalam local storage
localStorage.setItem('user', JSON.stringify(user));

// Mengambil object dari local storage
let userData = JSON.parse(localStorage.getItem('user'));
console.log(userData);

Latihan:

  • Buat form input nama dan tombol "Simpan" yang menyimpan data ke local storage.
  • Tampilkan data yang disimpan setiap kali halaman dimuat.
  • Tambahkan tombol "Hapus Data" untuk menghapus data dari local storage.
