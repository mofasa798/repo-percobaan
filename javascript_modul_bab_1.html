Modul Belajar JavaScript - Bab 1

Pengenalan JavaScript

1. Apa itu JavaScript?

JavaScript adalah bahasa pemrograman yang berjalan di sisi klien (browser) dan digunakan untuk membuat halaman web menjadi interaktif.

2. Sejarah Singkat JavaScript

  • Diciptakan oleh Brendan Eich pada tahun 1995.
  • Pada awalnya disebut Mocha, lalu berganti menjadi LiveScript, dan akhirnya menjadi JavaScript.
  • Didukung oleh hampir semua browser modern.

3. Menulis Script JavaScript

Script JavaScript bisa ditulis langsung dalam file HTML menggunakan tag <script>.

<!DOCTYPE html>
<html>
<head>
    <title>Belajar JavaScript</title>
</head>
<body>
    <script>
        alert('Hello World!');
    </script>
</body>
</html>

4. Menjalankan JavaScript

  • Melalui browser langsung
  • Melalui konsol browser (Inspect Element > Console)
  • Menggunakan platform Node.js (untuk backend)

Latihan:

Buat file HTML yang menampilkan alert "Hello World!" ketika halaman dibuka.
