Percabangan dan Perulangan

1. Percabangan (Conditional)

Percabangan digunakan untuk membuat keputusan berdasarkan kondisi tertentu.

If - Else

let nilai = 80;

if (nilai >= 75) {
    console.log('Lulus');
} else {
    console.log('Tidak Lulus');
}

Switch Case

let hari = 'Senin';

switch (hari) {
    case 'Senin':
        console.log('Hari kerja');
        break;
    case 'Sabtu':
    case 'Minggu':
        console.log('Hari libur');
        break;
    default:
        console.log('Hari tidak diketahui');
}

2. Perulangan (Looping)

For Loop

for (let i = 1; i <= 5; i++) {
    console.log('Angka ' + i);
}

While Loop

let i = 1;
while (i <= 5) {
    console.log('Angka ' + i);
    i++;
}

Do-While Loop

let i = 1;
do {
    console.log('Angka ' + i);
    i++;
} while (i <= 5);

Latihan:

  • Buat program untuk menentukan apakah sebuah angka ganjil atau genap.
  • Buat program untuk mencetak bilangan ganjil dari 1 sampai 100.
  • Buat program untuk menghitung grade siswa berdasarkan nilai:
    • 90-100: A
    • 80-89: B
    • 70-79: C
    • <70: D
