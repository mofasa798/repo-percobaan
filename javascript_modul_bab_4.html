Modul Belajar JavaScript - Bab 4

Fungsi

1. Apa itu Fungsi?

Fungsi adalah blok kode yang dapat digunakan berulang-ulang untuk menjalankan sebuah tugas tertentu.

2. Function Declaration

function sapa() {
    console.log('Halo, selamat datang!');
}

sapa(); // Memanggil fungsi

3. Function Expression

const sapa = function() {
    console.log('Halo, ini function expression!');
}

sapa();

4. Fungsi dengan Parameter dan Return

function luasPersegiPanjang(panjang, lebar) {
    return panjang * lebar;
}

let hasil = luasPersegiPanjang(5, 3);
console.log('Luas: ' + hasil);

5. Arrow Function

const luasSegitiga = (alas, tinggi) => {
    return (alas * tinggi) / 2;
};

console.log(luasSegitiga(4, 6));

Latihan:

  • Buat fungsi untuk menghitung keliling persegi panjang.
  • Buat fungsi untuk memeriksa apakah sebuah bilangan adalah bilangan prima.
  • Buat fungsi menggunakan arrow function untuk menghitung volume kubus.
