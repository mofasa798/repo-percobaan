Pengenalan Framework JavaScript: Node.js

1. Apa itu Node.js?

Node.js adalah runtime environment JavaScript yang memungkinkan menjalankan JavaScript di luar browser, umumnya digunakan untuk membuat backend aplikasi web.

2. Fitur Utama Node.js

  • Berbasis event-driven dan non-blocking I/O.
  • Cocok untuk aplikasi yang membutuhkan koneksi real-time.
  • Memiliki npm (Node Package Manager) untuk mengelola package.

3. Contoh Program Sederhana Node.js

// File: app.js
console.log('Halo dari Node.js!');

// Jalankan di terminal dengan perintah:
// node app.js

4. Membuat Server Sederhana dengan Node.js

// File: server.js
const http = require('http');

const server = http.createServer((req, res) => {
    res.writeHead(200, { 'Content-Type': 'text/plain' });
    res.end('Halo, ini server Node.js!');
});

server.listen(3000, () => {
    console.log('Server berjalan di http://localhost:3000');
});

Latihan:

  • Instal Node.js dan buat file JavaScript sederhana yang menampilkan pesan di terminal.
  • Buat server Node.js yang menampilkan pesan "Selamat Datang di Server Saya".
  • Uji server di browser menggunakan localhost:3000.
