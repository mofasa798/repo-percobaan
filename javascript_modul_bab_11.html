Asynchronous JavaScript (Async / Await)

1. Apa itu Asynchronous?

Asynchronous adalah proses menjalankan perintah yang tidak perlu menunggu perintah sebelumnya selesai, sehingga aplikasi tetap responsif.

2. Penulisan Async / Await

async function getData() {
    try {
        let response = await fetch('https://jsonplaceholder.typicode.com/posts');
        let data = await response.json();
        console.log(data);
    } catch (error) {
        console.error('Terjadi kesalahan:', error);
    }
}

getData();

3. Kelebihan Async / Await

  • Kode lebih mudah dibaca dan ditulis.
  • Mengurangi penggunaan callback yang terlalu dalam (callback hell).
  • Mudah dalam penanganan error dengan try...catch.

4. Contoh Menggunakan Async / Await

document.getElementById('tombol').addEventListener('click', async function() {
    let response = await fetch('https://jsonplaceholder.typicode.com/users');
    let users = await response.json();
    console.log(users);
});

Latihan:

  • Buat tombol untuk mengambil data dari API menggunakan async/await dan tampilkan di halaman.
  • Tambahkan penanganan error menggunakan try...catch.
  • Tambahkan loader "Memuat data..." saat proses fetch berlangsung.
