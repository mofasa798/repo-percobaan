Pengenalan Framework JavaScript: React JS

1. Apa itu React JS?

React adalah library JavaScript yang digunakan untuk membangun antarmuka pengguna (UI) yang interaktif dan dinamis dengan konsep komponen.

2. Konsep Dasar React

  • Component: Bagian UI yang dapat digunakan ulang.
  • JSX: Sintaks seperti HTML yang digunakan dalam React.
  • State: Data lokal pada component yang dapat berubah.
  • Props: Data yang dikirim ke component dari component lain.
