Array dan Object

1. Array

Array adalah kumpulan data yang disimpan dalam satu variabel.

let buah = ['Apel', 'Mangga', 'Pisang'];
console.log(buah[0]); // Apel

Metode Array

  • push() - Menambahkan data di akhir array
  • pop() - Menghapus data terakhir
  • shift() - Menghapus data pertama
  • unshift() - Menambahkan data di awal array
  • for loop - Menelusuri isi array
buah.push('Jeruk');
console.log(buah);

buah.pop();
console.log(buah);

2. Object

Object adalah struktur data yang berisi pasangan key dan value.

let siswa = {
    nama: 'Budi',
    umur: 17,
    jurusan: 'IPA'
};

console.log(siswa.nama);

Menambahkan dan Menghapus Properti

siswa.alamat = 'Bandung';
delete siswa.umur;
console.log(siswa);

3. Array of Object

let daftarSiswa = [
    { nama: 'Budi', umur: 17 },
    { nama: 'Ani', umur: 16 },
    { nama: 'Sari', umur: 18 }
];

console.log(daftarSiswa[1].nama); // Ani

Latihan:

  • Buat array berisi 5 jenis buah, lalu tampilkan satu per satu menggunakan for loop.
  • Buat object berisi data diri kamu (nama, umur, pekerjaan) lalu tampilkan semua datanya.
  • Buat array of object berisi daftar 3 buku (judul dan penulis) lalu tampilkan semua data buku tersebut.
