JavaScript Object-Oriented Programming (OOP)

1. Konsep OOP pada JavaScript

  • Class: Blueprint untuk membuat object.
  • Object: Instance dari class.
  • Constructor: Fungsi khusus untuk inisialisasi object.
  • Method: Fungsi yang dimiliki oleh object.

2. Membuat Class dan Object

class Mobil {
    constructor(merk, warna) {
        this.merk = merk;
        this.warna = warna;
    }

    info() {
        return `Mobil ${this.merk} berwarna ${this.warna}`;
    }
}

let mobil1 = new Mobil('Toyota', 'Merah');
console.log(mobil1.info());

3. Inheritance (Pewarisan)

class Kendaraan {
    constructor(nama) {
        this.nama = nama;
    }

    jalan() {
        return `${this.nama} sedang berjalan.`;
    }
}

class Motor extends Kendaraan {
    constructor(nama, tipe) {
        super(nama);
        this.tipe = tipe;
    }

    detail() {
        return `${this.nama} adalah motor tipe ${this.tipe}`;
    }
}

let motor1 = new Motor('Honda', 'Sport');
console.log(motor1.detail());
console.log(motor1.jalan());

Latihan:

  • Buat class Hewan dengan properti nama dan suara, serta method untuk menampilkan suara.
  • Buat class Kucing yang mewarisi class Hewan dan menambahkan properti warna.
  • Buat beberapa object dari class Kucing dan tampilkan datanya di console.
