Introduction to Fetch API (Pengambilan Data dari Server)

1. Apa itu Fetch API?

Fetch API adalah metode modern untuk mengambil data dari server secara asynchronous (tanpa me-refresh halaman).

2. Fetch Data dari API

fetch('https://jsonplaceholder.typicode.com/posts')
    .then(response => response.json())
    .then(data => console.log(data))
    .catch(error => console.error('Error:', error));

3. Menampilkan Data ke Halaman

fetch('https://jsonplaceholder.typicode.com/users')
    .then(response => response.json())
    .then(users => {
        let list = document.getElementById('userList');
        users.forEach(user => {
            let li = document.createElement('li');
            li.innerText = user.name;
            list.appendChild(li);
        });
    });

4. Penanganan Error

fetch('https://jsonplaceholder.typicode.com/posts/1')
    .then(response => {
        if (!response.ok) {
            throw new Error('Jaringan bermasalah');
        }
        return response.json();
    })
    .then(data => console.log(data))
    .catch(error => console.error('Terjadi error:', error));

Latihan:

  • Ambil data dari API https://jsonplaceholder.typicode.com/todos dan tampilkan di halaman.
  • Tambahkan penanganan error jika API gagal diakses.
  • Tambahkan loader (teks "Loading...") saat data sedang diambil.
