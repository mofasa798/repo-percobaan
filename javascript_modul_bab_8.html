Event Handling

1. Apa itu Event Handling?

Event handling adalah proses menangani interaksi pengguna dengan halaman web, seperti klik, input, hover, dll.

2. Jenis-Jenis Event

  • click: Saat elemen diklik
  • mouseover: Saat mouse berada di atas elemen
  • keydown: Saat tombol keyboard ditekan
  • submit: Saat form dikirim

3. Menambahkan Event Listener

let tombol = document.getElementById('tombol');
tombol.addEventListener('click', function() {
    alert('Tombol diklik!');
});

4. Prevent Default

Prevent default digunakan untuk mencegah perilaku bawaan elemen, seperti submit form yang me-refresh halaman.

let form = document.getElementById('form');
form.addEventListener('submit', function(e) {
    e.preventDefault();
    alert('Form berhasil ditangani tanpa reload!');
});

5. Event Bubbling

Event bubbling adalah proses dimana event naik dari elemen target ke elemen induknya.

document.getElementById('kotak').addEventListener('click', function() {
    alert('Kotak diklik!');
});

document.body.addEventListener('click', function() {
    alert('Body diklik!');
});

Latihan:

  • Buat tombol yang mengubah warna latar belakang setiap kali diklik.
  • Buat form sederhana dengan input nama, dan ketika disubmit, tampilkan alert dengan nama tersebut tanpa reload halaman.
  • Buat contoh sederhana event bubbling dengan 2 elemen yang bersarang, lalu tangani kedua event-nya.
