Pengenalan JavaScript Package Manager: npm

1. Apa itu npm?

npm (Node Package Manager) adalah manajer paket untuk JavaScript yang digunakan untuk mengelola library, tools, dan framework yang dibutuhkan dalam project.

2. Perintah Dasar npm

  • npm init: Membuat file package.json untuk mengelola project.
  • npm install <package>: Menginstal package.
  • npm uninstall <package>: Menghapus package.
  • npm install -g <package>: Menginstal package secara global.

3. Contoh Penggunaan npm

// Inisialisasi project npm
npm init -y

// Instalasi express
npm install express

// Menjalankan project
node app.js

4. Menjalankan Script dari package.json

// package.json
"scripts": {
    "start": "node app.js"
}

// Jalankan di terminal
npm start

Latihan:

  • Inisialisasi project menggunakan npm init.
  • Instal package express dan buat server sederhana.
  • Tambahkan script "start" di package.json untuk menjalankan server dengan perintah npm start.
