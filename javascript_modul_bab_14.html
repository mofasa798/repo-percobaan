Modular JavaScript (Menggunakan Module)

1. Pengertian Module

Module memungkinkan kita untuk memecah kode JavaScript menjadi beberapa file yang lebih kecil agar lebih terstruktur dan mudah dikelola.

2. Export dan Import

File: math.js

export function tambah(a, b) {
    return a + b;
}

export function kurang(a, b) {
    return a - b;
}

File: app.js

import { tambah, kurang } from './math.js';

console.log(tambah(5, 3));
console.log(kurang(5, 3));

3. Export Default

// File: greet.js
export default function() {
    console.log('Halo!');
}

// File: app.js
import greet from './greet.js';

greet();

4. Cara Menggunakan Module di Browser

Latihan:

  • Buat file module yang berisi fungsi penjumlahan, pengurangan, perkalian, dan pembagian.
  • Import dan gunakan semua fungsi tersebut di file utama.
  • Gunakan type="module" pada tag script HTML.
