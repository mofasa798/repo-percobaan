Pengenalan Testing pada JavaScript

1. Apa itu Testing?

Testing adalah proses untuk memastikan bahwa kode program berjalan sesuai dengan yang diharapkan dan bebas dari bug.

2. Jenis Testing

  • Unit Testing: Menguji fungsi atau komponen terkecil dari aplikasi secara terpisah.
  • Integration Testing: Menguji bagaimana beberapa bagian aplikasi bekerja bersama.
  • End-to-End (E2E) Testing: Menguji keseluruhan alur aplikasi seperti yang dilakukan oleh pengguna.

3. Contoh Unit Test dengan Jest

// sum.js
function sum(a, b) {
    return a + b;
}

module.exports = sum;
    
// sum.test.js
const sum = require('./sum');

test('menambahkan 1 + 2 sama dengan 3', () => {
    expect(sum(1, 2)).toBe(3);
});

4. Menjalankan Test dengan npm

// Tambahkan di package.json
"scripts": {
    "test": "jest"
}

// Jalankan perintah
npm test

Latihan:

  • Buat fungsi sederhana seperti penjumlahan atau pengurangan.
  • Buat file test untuk menguji fungsi tersebut menggunakan Jest.
  • Jalankan perintah npm test untuk melihat hasilnya.
