Form Handling dan Validasi

1. Mengambil Nilai Input

let inputNama = document.getElementById('nama');
document.getElementById('tombol').addEventListener('click', function() {
    alert('Nama Anda: ' + inputNama.value);
});

2. Validasi Form Sederhana

let form = document.getElementById('form');
form.addEventListener('submit', function(e) {
    e.preventDefault();
    let nama = document.getElementById('nama').value;

    if (nama === '') {
        alert('Nama tidak boleh kosong!');
    } else {
        alert('Form berhasil dikirim!');
    }
});

3. Validasi Panjang Karakter

let password = document.getElementById('password').value;
if (password.length < 6) {
    alert('Password minimal 6 karakter!');
}

4. Menampilkan Pesan Error di Halaman

let error = document.getElementById('error');
if (nama === '') {
    error.innerText = 'Nama wajib diisi.';
}

Latihan:

  • Buat form pendaftaran dengan input nama, email, dan password.
  • Validasi semua input tidak boleh kosong.
  • Validasi password minimal 6 karakter.
  • Tampilkan pesan kesalahan di halaman, bukan dengan alert.
