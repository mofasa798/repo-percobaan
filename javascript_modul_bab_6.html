DOM Dasar (Document Object Model)

1. Apa itu DOM?

DOM (Document Object Model) adalah representasi dari struktur halaman web yang memungkinkan JavaScript untuk mengakses dan memanipulasi elemen-elemen HTML.

2. Mengakses Elemen HTML

  • document.getElementById('id')
  • document.querySelector('selector')
let judul = document.getElementById('judul');
judul.style.color = 'blue';

let paragraf = document.querySelector('p');
paragraf.innerText = 'Teks ini telah diubah';

3. Mengubah Teks dan Style

let elemen = document.getElementById('contoh');
elemen.innerText = 'Ini adalah teks baru';
elemen.style.backgroundColor = 'yellow';

4. Event Listener Dasar

let tombol = document.getElementById('tombol');
tombol.addEventListener('click', function() {
    alert('Tombol diklik!');
});

Latihan:

  • Buat tombol yang ketika diklik, mengubah warna background halaman.
  • Buat paragraf yang teksnya berubah ketika diklik.
  • Buat input teks yang menampilkan hasilnya secara realtime di elemen lain (misalnya di heading).
